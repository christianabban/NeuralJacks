# NeuralJacks

Static website build for GitHub Pages.

## Repository Structure

```text
.
├── index.html
├── index_doubleclick.html
├── assets/
├── .nojekyll
└── README.md
```

## GitHub Pages Deployment

1. Upload these files to the root of your GitHub repository.
2. Open repository Settings > Pages.
3. Set Source to `Deploy from a branch`.
4. Select your branch, usually `main`, and folder `/root`.
5. Save and wait for GitHub Pages to publish.

`index.html` is the file GitHub Pages serves. `index_doubleclick.html` is included as a standalone local copy you can open directly on your computer.
